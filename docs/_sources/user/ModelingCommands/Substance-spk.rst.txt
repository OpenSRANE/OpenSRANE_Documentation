Substance Subpackage
********************

In this subpackage the materials and their properties are defined here by user. There are many properties that can be defined for the substances but all of them are not necessary for the models and depend on the models that user selected for the model, parameters or properties are needed. Most of the needed properties will be request in the warning file if are not defined by the user.
Also, there is DataBank module with predefined materials that can be complete more simply in the future. The available commands of this subpackage are described in the following:

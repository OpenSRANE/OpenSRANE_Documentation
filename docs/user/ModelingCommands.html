Modeling commandsÔÉÅ

In this section, all the commands that should be used to make an OpenSRANE model has been described. The common way of using the following commands has been shown in the Modeling Steps.

Note

OpenSRANE mainly created from Subpackages that each subpackage is responsible for something different from other subpackages. Each subpackage contain commands of OpenSRANE and in this each bundle of OpenSRANE commands are separated from the others. So, in the following we see the subpackages name and inside each subpackage the OpenSRANE commands are presented.

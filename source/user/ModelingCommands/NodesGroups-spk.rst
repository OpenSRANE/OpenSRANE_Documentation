NodesGroups Subpackage
**********************

In this subpackage user can define a set of nodes for various purposes. Mainly they are used for defining any vulnerable areas like (populations, green areas, roads and ...) object. These objects accept assigning defined probit objects and according assigned probits, their vulnerability will be calculated.
